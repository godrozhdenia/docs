# wallet manager

manage all your solana wallets from one place.

## create wallets

tap **wallets** > **create wallet** to generate a new wallet instantly.

your wallet address is displayed immediately and ready to receive funds.

## multiple wallets

you can create multiple wallets based on your tier:

| tier | max wallets |
|------|-------------|
| free | 5 |
| bronze | 25 |
| silver | 50 |
| gold | 100 |
| diamond | unlimited |

## import existing wallets

have an existing wallet? import it:

1. tap **wallets** > **import**
2. paste your private key
3. done - it's added and encrypted

your original message with the key is automatically deleted for security.

## export your keys

backup your private keys anytime:

1. tap **wallets**
2. select a wallet
3. tap **export key**

the key shows briefly then auto-deletes. save it somewhere safe.

## security

- all keys are encrypted instantly when created or imported
- we cannot see your private keys
- keys are only decrypted when you sign a transaction
- export and backup your keys regularly

## wallet uses

- **primary wallet** - used for single trades
- **swarm wallets** - created automatically for multi-wallet buys
- **volume wallets** - used for volume generation sessions

all wallets are managed from the same wallets menu.
